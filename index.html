UV Competition Database Matcher

Find matching UV systems based on your project parameters

📊 Option A — Google Sheets (Recommended)

  1. Open your UV Competition Database spreadsheet in Google Sheets
  2. Click File → Share → Share with others
  3. Under "General access", change to "Anyone with the link"
  4. Set the role to Viewer and click Done
  5. Click Copy link — it will look like:
    https://docs.google.com/spreadsheets/d/1ABC...XYZ/edit?usp=sharing
  6. Paste that link into the "Google Sheets / Online URL" field
  7. The app automatically converts it to a CSV export URL
⚠️ CORS Warning: Google Sheets fetching only works when served from a web server, NOT from file:///...
✅ Tip: For multiple tabs, add &gid=XXXXXXX to the URL.

📁 Option B — Local Excel or CSV File

  1. Export or save your database as .xlsx, .xls, or .csv
  2. Make sure Row 1 contains the column headers
  3. Click "Choose File" under the Local File section
  4. Select your file and click Search
✅ Supported formats: .csv, .xls, .xlsx — works offline from your desktop.
⚠️ For .xls/.xlsx files, SheetJS is loaded from CDN — internet required on first use.

🌐 Option C — Host the App

  1. GitHub Pages: Upload HTML → Settings → Pages → Deploy
  2. Netlify: Drag & drop at netlify.com/drop
  3. VS Code Live Server: Right-click → "Open with Live Server"
  4. Python: python -m http.server 8080 then open http://localhost:8080

🗂️ Required Column Headers

  • Date, Country, Region, Project Name
  • Application, Open Channel/Closed Vessel
  • Competitor, Model, Qty. of Units
  • Flow Rate [MGD], Flow Rate [m³/h]
  • UVT [%/1 cm], UV Dose [mJ/cm²], Type of Dose
  • No. of Channels, No. of Lamps Total, No. of Lamps Duty
  • Power Consumption [kW], Price [U$], Price per Lamp [U$ / lamp]
  • Comments
✅ The app uses fuzzy column matching — minor header variations and Excel newlines are handled automatically.
